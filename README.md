## :wrench: Technologies and methods used

- React Native;
- NodeJS
- MySQL
- UML
- SCRUM
- C4 Model
- Figma
- Trello
- Eslint
- Prettier

## :rocket: Running the project

Suggestion: you must have node js installed on your machine and install node modules

```
git clone https://github.com/Molezinif/Ordo.git
cd .\app
npm start
```
